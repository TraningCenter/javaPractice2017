# LiftProj

## Копиляция и запуск
Чтобы скомпилировтаь программу, необходимо выполнить команду:
mvn clean install
Для того, чтобы запустить приложение, выполните:
java -jar LiftProj-1.0.jar

## Действия пользователя
После запуска приложения пользователю предлагают выбрать одно из четырех действий. 
Первые три позволяют ему использовать готовый набор входных данных.
Четвертый вариант попросит  пользователя задать свои параметры системы: 
количесвто этажей в доме, количество лифтов и тд. 

## Последовательность действий приложения
1. Пользователь выбирает каким обрзом задаются даные
2. Данные задаются
3. Программа просчитывает все действия
4. Программа отрисовывает все действия на консоль
5. Выполнение программы завершается

## О визуализации
Приложение рисует все лифты, всех людей и все кнопки, обновляя картинку 2 раза в секунду. 
О пассажире указывается его номер, метка 'P' (Passenger) и направление, в котором он хочет ехать.
Для кнопок отображается состояние: выключена (0) и включена (!).
ДЛя лифтов отображается номер и метка 'L' (Lift). Открытие и закрытие дверей лифта отображается с помощью символов <> и >< соответсвено.
